Reference Guide for Hub

Class Function
container Adds 16 pixels of padding
margin Adds 16 pixels of margin
text-center Centers text (previously called center)
card Adds a shadow to the element
card2 Adds a shadow to the element (thicker shadow)
round Makes the border radius 5 pixels all sides
round2 Makes the border radius 10 pixels all sides
float-left Makes the element float to the left
float-right Makes the element float to the right
text-left Makes the text go left
text-right Makes the text go right
font-18 Makes the text 18 pixels big
font-14 Makes the text 14 pixels big
text-underline Makes the text have an underline
text-bold Makes the text bold (Previously called bold)
border-left Makes the element have a left border
